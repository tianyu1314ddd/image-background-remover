## 一、产品概述

### 1.1 产品定位
一款在线 AI 图片背景去除工具，用户上传图片后自动去除背景，支持下载透明背景或替换为纯色背景的图片。

### 1.2 目标用户
- 电商卖家（商品主图处理）
- 设计师（快速抠图）
- 普通用户（证件照、社交媒体头像处理）

### 1.3 核心价值
- **快**：上传即处理，秒级出结果
- **准**：Remove.bg API，业界顶级抠图精度
- **简**：无需注册，无需安装，打开即用

| 层级 | 技术选型 | 说明 |

## 三、功能需求

### 3.1 核心功能（MVP 必须）

#### F1 — 图片上传
- 支持点击上传和拖拽上传
- 支持格式：JPG、PNG、WebP
- 文件大小限制：≤ 25MB（Cloudflare Workers 免费版限制）
- 上传后立即触发处理，无需额外点击

#### F2 — 背景去除
- 调用 Remove.bg API 处理图片
- 处理过程显示 Loading 状态（进度条或动画）
- 处理失败显示友好错误提示（如 API 额度不足、图片格式不支持）

#### F3 — 结果预览
- 左右对比展示：原图 vs 去背景结果
- 支持棋盘格背景（透明区域可视化）
- 支持切换预览背景色（透明 / 白色 / 黑色）

#### F4 — 图片下载
- 下载格式：PNG（保留透明通道）
- 下载文件名：`{原文件名}-removed-bg.png`
- 一键下载，无需登录

#### F5 — 重新上传
- 下载后或处理后，支持一键清空，上传新图片

### 3.2 增强功能（MVP 可选，视时间决定）

#### F6 — 自定义背景色
- 提供颜色选择器，用户可选择任意纯色作为背景
- 预设常用颜色：白、黑、红、蓝、绿

#### F7 — 批量处理（MVP 后迭代）
- 一次上传多张图片
- 逐张处理，显示整体进度

## 五、API 设计

### POST `/api/remove-bg`

**请求**
```
Content-Type: multipart/form-data
Body: image (File)
```

**响应（成功）**
```json
{
"success": true,
"imageBase64": "data:image/png;base64,..."
}
```

**响应（失败）**
```json
{
"success": false,
"error": "错误描述"
}
```

**处理流程**
1. 接收前端上传的图片（内存，不写磁盘）
2. 将图片 Buffer 转发给 Remove.bg API
3. 接收 Remove.bg 返回的透明 PNG
4. 将结果转为 Base64 返回前端
5. 前端直接渲染和下载，全程无存储

## 六、非功能需求

### 6.1 性能
- 首屏加载 < 2s（Cloudflare CDN 加速）
- 图片处理响应 < 10s（受 Remove.bg API 速度影响）

### 6.2 隐私安全
- 图片不存储到任何服务器
- API Key 仅在服务端使用，不暴露给前端
- 页面添加隐私说明："您的图片不会被存储"

### 6.3 兼容性
- 支持现代浏览器：Chrome、Safari、Firefox、Edge
- 移动端响应式适配

### 6.4 SEO
- 页面 Title：`Free Image Background Remover - Remove BG Online`
- Meta Description：`Remove image background online for free. No signup required. Powered by AI.`
- 结构化数据（WebApplication schema）

## 七、错误处理

| 错误场景 | 用户提示 |
|----------|----------|
| 文件格式不支持 | "请上传 JPG、PNG 或 WebP 格式的图片" |
| 文件超过 25MB | "图片大小不能超过 25MB" |
| Remove.bg API 额度耗尽 | "今日处理次数已达上限，请明天再试" |
| 网络超时 | "处理超时，请检查网络后重试" |
| 其他 API 错误 | "处理失败，请稍后重试" |
